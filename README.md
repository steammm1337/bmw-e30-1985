# BMW E30 1985 White - Интерактивный сайт

Интерактивный 3D сайт-презентация легендарного BMW E30 1985 года в белом цвете.

## Особенности

- **3D модель BMW E30** с возможностью вращения
- **Скролл-анимация** — модель крутится при прокрутке страницы (360° на экран)
- **Видео-переход** — когда модель повернута задом, плавно появляется видео на 5 секунд
- **Orbital controls** — управление камерой мышью/тачем
- **BMW брендинг** — фирменные синие/белые цвета, неоновые акценты
- **Гаражный стиль** — темный фон, бетон, сетка, рим-лайты
- **Полная адаптивность** — работает на десктопе и мобильных
- **Информационные секции** — технические характеристики, история, культурное значение

## Как работает

### Скролл-механика
- Hero секция высота = 200vh (два экрана)
- При скролле модель плавно вращается на 360°
- Rotation = scrollProgress × 2π

### Видео-переход
- Когда модель повернута на ~180° (багажник видно)
- И scrollProgress между 40-60%
- Видео появляется на 5 секунд
- Затем плавно исчезает, модель продолжает вращаться

### Освещение
- Ambient light (базовая подсветка)
- Directional light (солнце, тени)
- 2× Point lights (BMW синий рим-лайт)
- Spotlight сверху

### Оптимизация
- `setPixelRatio(Math.min(devicePixelRatio, 2))` — макс 2x
- Shadows: PCFSoft 2048×2048
- Mobile: pixelRatio = 1
- Damping для плавности

## Кастомизация

### Изменить цвета BMW
В `style.css`:
```css
:root {
    --bmw-blue: #1c69d4;        /* Основной синий */
    --bmw-light-blue: #4a9eff;  /* Светлый синий */
}
```

### Скорость вращения
В `main.js`:
```js
const ROTATION_RANGE = Math.PI * 2; // 360° = 2π
```

### Время видео

```js
if (videoPlaying && Date.now() - videoStartTime > 5000) { // 5 секунд

```
